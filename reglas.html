🧭 Reglas de decisión y tablas de umbrales

Criterios accionables extraídos del examen — para fijar lo que ya sabés y reforzar lo que falló.

0 · Principios transversales (heurísticas)

✗ Emergencia → estabilizar (ABC) ANTES de diagnosticar o intervenir. Volumen/vía aérea/calcio antes que endoscopía, TC o procedimiento.Q46, Q53, Q63
✗ Cuadro estable → primera línea / lo menos invasivo / step-up. La opción heroica (cirugía abierta, ICP, biopsia) suele ser distractor si hay conducta médica inicial válida.Q11, Q32, Q65, Q89
✗ Banderas rojas → nunca es funcional. Anemia, baja de peso, sangre, síntomas nocturnos, VSG/PCR altas → estudiar organicidad.Q20
✓ Leé la consigna exacta: "inicial" ≠ "más adecuada" ≠ "que confirma" ≠ "con intención curativa". Cambia la respuesta.Q16, Q30
✓ Esquema de vacunas atrasado → completar, NO reiniciar.Q71

1 · Cardiología

✗ Angina estable + función VI normal → tratamiento médico óptimo (β-bloq + estatina + AAS). La ICP no mejora sobrevida (COURAGE/ISCHEMIA).Q65
✗ FA aguda estable → control de frecuencia (betabloqueante) + anticoagulación. Cardioversión solo si inestable.Q41
✗ Prevención secundaria (post-IAM) → estatina de alta intensidad siempre, independiente del LDL.Q49
✗ HTA grado 2 que no responde a medidas → combinar 2 fármacos a dosis bajas. Nunca IECA + ARA II.Q61
✗ HTA oculta = consultorio normal + ambulatorio alto. Guardapolvo blanco = lo opuesto.Q59
✓ SCASEST: angina in crescendo / de reposo + cambios dinámicos de onda T/ST sin elevación persistente.Q18
✓ FA con necesidad de anticoagular y difícil control de INR (zona rural) → DOAC (no requiere monitoreo).Q36
✓ Dx de HTA: confirmar en ≥2 consultas, promedio ≥140/90.Q44
✓ Endocarditis (Duke): hemocultivos ×3 + ETT + ATB dirigida.Q38
✓ Taponamiento: tríada de Beck + pulso paradójico + bajo voltaje/alternancia. TEP masivo con shock refractario → trombolíticos.Q57, Q51
✓ Soplo diastólico = siempre patológico → ecocardiograma.Q97

2 · Neumonología

✗ Patrón UIP / FPI → antifibróticos (pirfenidona/nintedanib). Corticoide+azatioprina contraindicado (PANTHER).Q15
✗ Derrame paraneumónico no complicado: exudado neutrofílico, pH>7,2, glucosa normal. Complicado si pH<7,2 o glucosa<60.Q76
✓ Asma (GINA) → iniciar con corticoide inhalado de entrada, no SABA solo.Q8
✓ EPOC con ≥2 exacerbaciones/año → LAMA + LABA. CI solo con eosinofilia/asma.Q21
✓ NAC sin criterios de gravedad (CURB-65 bajo) → ambulatorio con amoxicilina.Q52

3 · Gastroenterología

✗ "SII" con banderas rojas (diarrea nocturna, anemia, VSG↑) → videocolonoscopia, no tratamiento sintomático.Q20
✗ Ascitis cirrótica (GASA ≥1,1) → restricción de Na + espironolactona. Furosemida se agrega; restricción hídrica solo si hiponatremia.Q26
✗ HDA con shock → reanimar (cristaloides + tipificar/reservar) antes de endoscopía.Q46
✓ Tamizaje CCR con familiar de 1er grado → colonoscopía a los 40 años (o 10 años antes del caso índice), repetir c/5 años.Q24
✓ Sangrado rectal con rectosigmoideoscopía normal → videocolonoscopía completa.Q94

4 · Nefrología y medio interno

✗ Hiperkalemia con cambios en ECG → gluconato de calcio IV primero (estabiliza membrana). Luego insulina+dextrosa; diálisis es definitiva, no inmediata.Q53
✓ Hiponatremia: clasificar por volemia (hipovolémica vs SIADH vs hipervolémica) + osmolaridad.Q2
✓ Acidosis metabólica: calcular anión gap. AG normal + diarrea = hiperclorémica.Q67
✓ ERC + DBT + albuminuria → SGLT2 + IECA/ARA II + suspender AINE.Q42
✓ Pielonefritis que no mejora a las 72h → TC (buscar absceso/obstrucción) + hemocultivos + ampliar ATB.Q31

5 · Endocrinología

✓ Cetoacidosis diabética → reposición con SF primero (volumen), después insulina; vigilar K.Q5
✓ Coma mixedematoso (TSH↑, hipotermia, hipoNa, hipercapnia) → levotiroxina (±T3) + hidrocortisona + soporte.Q73
✓ Hipotiroidismo: etiología más probable → anti-TPO (Hashimoto).Q82
✓ DM2 con HbA1c muy alta (≥10%) + síntomas → metformina + insulina basal.Q95

6 · Infectología

✓ Sospecha de TBC → prueba molecular rápida (GeneXpert) + evaluar contactos estrechos.Q10
✓ TBC latente (contacto, sobre todo VIH+) tras descartar activa → isoniacida preventiva.Q78
✓ Sepsis (lactato ≥4) → prioridad: cristaloides + hemocultivos + ATB precoz (bundle 1h).Q63
✓ VIH con CV detectable persistente y buena adherencia → test de resistencia.Q99
✓ Adulto sin registros de vacunas → antigripal + antineumocócica + iniciar Td (no reiniciar lo que sí tenga).Q70

7 · Neurología

✓ ACV isquémico <4,5h + TC sin hemorragia → trombólisis con alteplase.Q45
✓ Compresión medular (déficit + dolor + nivel) → dexametasona + RM de columna total (urgencia).Q12
✓ HSA día 4–14 con deterioro → isquemia tardía por vasoespasmo (nimodipina profiláctica).Q86
✓ Parálisis facial periférica (Bell): compromete la frente. Central = respeta la frente.Q91
✓ Demencia vascular: deterioro escalonado + factores/eventos vasculares. Delirium: agudo, fluctuante, falla la atención → buscar causa.Q40, Q88

8 · Hematología

✓ Ferropenia (VCM↓, ferritina↓, TIBC↑, sat↓) sin alarma → hierro oral; con sangrado/edad → estudiar.Q7
✓ SAF: trombosis + abortos + TTPa prolongado → anticoagular + anticardiolipina/anti-β2GPI/anticoagulante lúpico.Q19
✓ LMC: leucocitosis con desviación a izquierda + basofilia + esplenomegalia (BCR-ABL).Q100

9 · Pediatría

Agudo / emergencias
✗ Anafilaxia → adrenalina IM + O₂ (EV solo en shock refractario monitorizado).Q39
✗ Bronquiolitis con SatO₂<92–93% / ↓ingesta → O₂ y soporte. Sin ATB ni broncodilatadores de rutina.Q66
✗ Crup leve → corticoide oral (dexametasona) igual. Adrenalina NBZ solo si estridor en reposo.Q77
✗ ITU → urocultivo + ATB empírico sin demorar. Eco no de rutina en 1er episodio.Q72
✗ Anemia severa (Hb 7) con hipoxemia → transfundir. Hierro oral = ferropenia estable.Q84
Crónico / control de salud
✗ Chupete tras lactancia establecida → ofrecer al inicio de cada sueño (protege de muerte súbita).Q68
✗ Cojera post-catarro (sinovitis transitoria) → ecografía de cadera. Artrocentesis solo si sospecha séptica.Q90
✗ Prurito anal nocturno + gusanos blancos ~1cm → enterobiasis (oxiuros).Q96
✓ Estenosis hipertrófica de píloro: 3–6 sem, vómito no bilioso en proyectil, oliva epigástrica, alcalosis hipoclorémica → eco.Q1
✓ OMA (tímpano abombado) → analgesia + amoxicilina. Impétigo = costras melicéricas. Varicela = lesiones en distintos estadios.Q3, Q48, Q98
✓ Enuresis primaria monosintomática → medidas conductuales + alarma (1ª línea). Genu varo <2–3 años = fisiológico.Q23, Q55
✓ Sueño seguro: supino, superficie firme, habitación compartida sin colecho.Q58
✓ Apendicitis pediátrica con peritonismo → ayuno + hidratación + ATB + cirugía (no más estudios si la clínica es clara).Q22

10 · Cirugía

✗ Apendicitis en adulto: el estudio que confirma = TC con contraste (no eco).Q16
✗ Pancreatitis necrosante infectada → drenaje percutáneo + ATB (step-up); necrosectomía abierta = último escalón.Q32
✗ HBP con PSA estable → α-bloqueante; biopsia solo si PSA/tacto sospechosos.Q11
✗ Diverticulitis aguda → TC con contraste. Colonoscopía contraindicada en agudo (a las ~6 semanas).Q89
✓ Colecistitis/colangitis (Murphy+, fiebre, leucocitosis) → ayuno + hidratación + ATB + IC quirúrgica.Q25
✓ Neumotórax hipertensivo (desviación traqueal, IY, ausencia MV) → descompresión inmediata, sin esperar Rx.Q54
✓ Obstrucción intestinal por bridas (cicatriz) → SNG + hidratación + Rx + IC cirugía. Hernia reducible → reparación electiva.Q79, Q56
✓ Ca de páncreas de cabeza resecable → duodenopancreatectomía cefálica (Whipple).Q30

11 · Ginecología y obstetricia

✗ Alto riesgo de mama (BRCA, RT torácica 10–30a) → RM mamaria con contraste anual.Q9, Q60
✗ Densitometría: adelantar (antes de 65) si hay factores de riesgo (posmenopausia + tabaco + bajo peso).Q80
✗ Post-tratamiento de HSIL → vacunar VPH (reduce recurrencia).Q6
✗ Sífilis en embarazo: título estable a 6 sem ≠ falla; falla = aumento ≥4× (2 diluciones) → seguir serología.Q13
✓ Anticoncepción en puerperio con lactancia → progestágeno solo (sin estrógenos por riesgo trombótico).Q64, Q69
✓ EPI: dolor a la movilización cervical + secreción. Ca de endometrio → histerectomía + anexectomía + estadificación.Q81, Q92
✓ Parto prematuro 24–34 sem → tocólisis + corticoides antenatales + vigilancia.Q43
✓ Preeclampsia severa → labetalol + sulfato de magnesio. HTA gestacional = ≥140/90 tras 20 sem, sin proteinuria.Q74, Q85

12 · Salud pública, epidemiología y bioética

✗ Aleatorización → comparabilidad basal (confundidores). Enmascaramiento → sesgo de medición.Q75
✓ No perder casos graves (cribado) → priorizar sensibilidad.Q35
✓ Diseños: prevalencia + asociación en un momento = corte transversal.Q29
✓ Paciente competente puede rechazar un tratamiento (Ley 26.529): respetar y dejar constancia.Q62
✓ Parto humanizado (Ley 25.929): acompañamiento y fisiología en trabajo de parto, parto y postparto.Q50
✓ PROA: permitir ATB restringido 24–48h y luego exigir autorización del equipo (no frenar el inicio).Q47
